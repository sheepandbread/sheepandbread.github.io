简介

JavaScript是一种具有函数有限特性的轻量级、解释型（即时编译型[just-in-time compiling]）编程语言。

进一步说，JavaScript 是一种基于原型、多范式、单线程的动态语言，并且支持面向对象、命令式和声明式（如函数式编程）风格。

（严格来说，JavaScript并不算面向对象编程语言。ES6后新增了class，使它更贴合传统的oop。但在Netscape官方定义，JavaScript是基于对象的语言。）

JavaScript是Netscape网景公司的产品（而java是sun家的，现在是Oracle）。

JavaScript没有Java的静态类型和强类型检查特性，但JavaScript 遵循了 Java 的表达式语法，命名规范以及基础流程控制。

JavaScript Java
面向对象。不区分对象类型。通过原型机制继承，任何对象的属性和方法均可以被动态添加。 基于类系统。分为类和实例，通过类层级的定义实现继承。不能动态增加对象或类的属性或方法。
变量类型不需要提前声明 (动态类型)。 变量类型必须提前声明 (静态类型)。
不能直接自动写入硬盘。 可以直接自动写入硬盘。

Js是弱类型，Java是强类型。

  • 强类型就是定义变量的时候，类型已确定，并且不可变。

  • 弱类型就是类型可变。

JavaScript的标准化组织是ECMA。遵循该标准化版本的 JavaScript 被称作 ECMAScript。

特点：

  1. 交互性（它可以做的就是信息的动态交互）
  2. 安全性（不允许直接访问本地硬盘）
  3. 跨平台性（只要是可以解释Js的浏览器都可以执行，和平台无关）

引入方式：

  • 内部：在head或body中，使用script标签来书写。

  • 外部：外部文件引入。（这样更有序，更易于复用）

注释：

  • defer：脚本将按照页面出现的顺序加载。在页面内容全部加载完毕之前，脚本不会运行。

1.基础

1.1语法和数据类型

JavaScript 的基本语法、变量声明、数据类型和字面量（literal）

JavaScript区分大小写，并使用Unicode字符集。

指令被称为语句，用;分隔。

单行注释：//，多行注释：/* */。

声明

JavaScript有三种声明方式：

  • var：声明一个变量
  • let：声明一个块作用域的局部变量
  • const：声明一个块作用域的只读常量

若声明时未赋值，则其值为undefined。

在 数值类型环境 中，undefined会转换为NaN，而null会被当成0（在布尔环境下被当成false）。

访问未声明的变量会抛出ReferenceError异常。

变量名应当使用拉丁字符 (0-9,a-z,A-Z) 和下划线字符。

  • 变量名大小写敏感
  • 变量名不能以数字开头（以字母、下划线（_）或美元符号（$）开头）
  • 变量名避免使用js的保留字（如var、function、let、for等）

变量的作用域：全局变量、局部变量

  • 在函数之外声明的变量——全局变量。可被文档任何其他代码访问。

    在网页中，全局对象是window[1]。所以可以使用如windows.*variable*的语法来设置和访问全局变量。

  • 在函数内部声明的变量——局部变量。只可在当前函数的内部使用。

先使用变量，后声明——变量提升。

  • var输出undefined
  • 在ECMAScript6中，let、const在声明前引用会抛出错误（Reference）。

函数也可进行函数提升（函数表达式不可）。

常量的命名规则和变量相同（详见变量命名规则）。

在统一作用域内，不能使用和变量名或函数名相同的名字来命名常量。

对象属性和数组被赋值/定义为常量时是不受保护的。

数据结构和类型

目前的ECMAScript标准定义了8种数据类型：

    • 任意精度的整数BigInt。安全地存储和操作大整数。
    • 字符串String。一串表示文本值的字符序列。
    • 代表Symbol。ES6新增类型。一种实例是唯一且不可改变的数据类型
  • 对象Object

  • 数字转换为字符串

    • 使用加法运算符+。（涉及其他运算符不会转为字符串）

  • 字符串转换为数字

    • parseInt() ——该方法只能返回整数

    • parseFloat()

    • 使用一元加法运算符

字面量literal：脚本中按字面意思给出的固定的值。

  • 数字字面量

    • 整数字面量。例如十进制、十六进制（0x或0X开头）、八进制（ 0或 0O或0o开头）以及二进制（0b或0B开头）。

1.2流程控制与错误处理

JavaScript 提供一套灵活的语句集，特别是控制流语句，你可以用它在你的应用程序中实现大量的交互性功能。

语句块：用一对大括号{}界定。

  • 语句块常用于流程控制，如if,for,while等。
  • ES6之前没有块作用域。之后使用let和const是块作用域的。

条件判断

  • if…else语句

    判断条件真假。

    • 如果条件表达式中使用赋值，通常在赋值语句前后额外加一对括号

    • 会被计算为false的值

      • false
      • undefined
      • null
      • 0
      • NaN
      • 空字符串（""）

  • switch语句

    匹配表达式的值。

异常处理

可以用throw语句抛出一个异常，并用try…catch语句捕获处理它。

  • throw语句

    使用throw语句抛出一个异常。

    需要规定一个含有值的表达式被抛出。

  • try…catch语句

    抛出异常时，try…catch语句捕获它。

    • catch块用于处理try块中产生的异常。
    • finally块无论是否抛出异常都会执行。

  • error对象

    Error() 构造函数能够创建一个包含错误信息的对象。

    根据错误类型，可以使用error构造函数，用参数提供更精炼的信息。

JavaScript 中提供了这些循环语句：

for 语句、do…while 语句、while 语句、label 语句、break 语句、continue 语句、for…in 语句、for…of 语句

  • for

    过程：

    1. 若有initialExpression，执行。通常该表达式会初始化循环计数器。
    2. 计算condition表达式的值，true则循环，fasle循环终止
    3. 执行statement
    4. 若有incrementExpression，执行
    5. 回到步骤2

    一直执行直到condition为假。

  • while

    condition为真就一直执行。

  • label

    一个位置标识符。可以用label表示一个循环，然后使用break或continue来控制循环继续或停止。

  • break

    用于终止循环、switch或链接到label语句。

    继续执行（跳过代码块剩余部分进行下一循环）一个while、do-while、for或label语句。

    • 数组更建议使用传统的for循环迭代。

  • for…of

    语句在可迭代对象（包括Array、Map、Set、argument等）上创建一个循环，对值的每一个独特属性进行一次迭代。

1.4函数

函数是 JavaScript 中的基本组件之一。

JavaScript 中的函数类似于过程——一组执行任务或计算值的语句。

但要成为函数，这个过程应该接受输入并返回与输入存在某些明显关系的输出。

要使用一个函数，你必须将其定义在你希望调用它的作用域内。

函数声明

一个函数定义（也叫函数声明）由function关键字，和以下部分组成：

  • 函数名称
  • 函数参数列表（括号内用逗号分隔）
  • 定义函数的JavaScript语句。

参数本质上是按值传递给函数。所以函数体的代码为传递给函数的参数赋了新值，这个改变也不会反映到全局或调用该函数的代码中。

嵌套函数和闭包

可以在函数里嵌套另外一个函数。

它的自身形成一个闭包。

  • 闭包是可以拥有独立变量以及绑定了这些变量的环境（“封闭”了表达式）的表达式（通常是函数）
  • 当内部函数以某一种方式被任何一个外部函数之外的任何作用域访问时，就会创建闭包。

总结：

  • 内部函数只能在外部函数中访问
  • 内部函数形成闭包，他可以访问外部函数的参数和变量，但外部函数不能使用它的参数和变量。

当统一作用域下两个参数或者变量同名时，会产生命名冲突。最近的优先级最高，最远的最低。

预定义函数

  • isFinite() ：用来判断被传入的参数值是否为一个有限数值（finite number）

    • 参数为Nan或正负无穷大——返回false
    • 其他——true

  • isNaN() ：判断一个值是否是 NaN。

    给定值返回true为NaN，否则为false。

    • NaN：表示一个非数字的值。
      • NaN值的产生：算术运算返回一个未定义或无法表示的值时。例如0除以0（但其他数值除以0不会产生NaN）
    • 如果isNaN函数的参数不是Number类型，会先强转为数值后进行判断。
      • 但对于能强转为有效的非NaN数值来说，返回false就显得很奇怪。例如不是NaN的空字符串和布尔值，会被强制转换为数值0和1。
    • Number.isNaN() 静态方法比**isNaN()**更加健壮。

  • parseFloat() ：解析一个参数（必要时先转换为字符串）并返回一个浮点数。

  • **parseInt()**：解析字符串参数，并返回指定的基数的整数。

  • decodeURI() ：对先前经过 encodeURI函数或者其他类似方法编码过的统一资源标志符（URI）进行解码。

  • encodeURI() ：通过将特定字符的每个实例替换为一个、两个、三或四转义序列来对统一资源标识符 (URI) 进行编码。

  • decodeURIComponent() 方法对先前经过 encodeURIComponent函数或者其他类似方法编码的统一资源标志符（URI）进行解码。

  • encodeURIComponent() 方法通过以表示字符的 UTF-8 编码的一个、两个、三个或四个转义序列替换统一资源标识符（URI）的某些字符来进行编码。

  • escape() ：方法生成一个新的字符串，其中的某些字符已被替换为十六进制转义序列。

    • 已不再推荐。请使用 encodeURI() 或 encodeURIComponent() 代替。

  • unescape() 方法计算生成一个新的字符串，其中的十六进制转义序列将被其表示的字符替换。

    • 已不再推荐。请使用 decodeURI() 或 decodeURIComponent() 替代。

运算符

赋值运算符

将右边操作数的值赋给左边的操作数。

最简单的赋值运算符就是=，例如x=y，就是把y的值赋给x。

还有一些复合赋值操作符，例如：

  • 加+、减-、乘*、除/、求余%、求幂**
  • 左移位运算<<，右移位运算>>
    • 将一个操作数的二进制表示形式向左/右移动指定位数。
  • 无符号右移位赋值>>>
  • 按位与&、按位异或^、按位或|

解构赋值：将数组中的值或对象的属性取出，赋值给其他变量。

比较运算符

比较它的操作数并返回一个基于表达式是否为真的逻辑值。

在多数情况下，如果两个操作数不是相同的类型，JavaScript 会尝试转换它们为恰当的类型来比较。

  • 等于 ==：两边操作数相等返回true
  • 不等于!=：两边操作数不相等返回true
  • **全等===**：两边操作数相等且类型相同时返回true。
  • **不全等!==**：两边操作数不相等 或 类型不同时返回ture
  • 大于>、大于等于>=
  • 小于<、小于等于<=

算术运算符

算术运算符使用数值 (字面量或者变量) 作为操作数并返回一个数值。

标准的算术运算符就是加减乘除 (+ - * /)。

其他的js算术运算符：

  • 求余%
  • 自增++、自减–
  • 一元负值符-：返回操作数的负值
  • 一元正值符+：如果操作数在之前不是数值，尝试转换为数值。
  • 指数运算符**：计算底数（base）和指数（exponent）次方。表示为base^exponent。

位运算符

位运算符将它的操作数视为 32 位元的二进制串。（而非其他进制）

  • 按位与AND：a&b，都为1则为1
  • 按位或OR：a|b，有一位1则为1
  • 按位异或XOR：a^b，仅有一位1时为1
  • 按位非NOT：~a
  • 左移shift：a<<b，将a的二进制数表示 向右移动b位，丢弃被移出的位。
  • 算术右移：a>>b。左移，空位根据最高位是0或是1来填充。
  • 无符号右移：a>>>b。

逻辑运算符

逻辑运算符常用于布尔（逻辑）值之间。

  • 逻辑与&&
  • 逻辑或||
  • 逻辑非 !expr

短路求值：作为逻辑表达式进行求值是从左到右，它们是为可能的“短路”的出现而使用以下规则进行测试。

  • false && anything // 被短路求值为 false
  • true || anything // 被短路求值为 true

字符串运算符

用连接操作符(+) 将两个字符串值相连接。

也可以简写为+=来拼接。

一元运算符

仅对应一个操作数。

  • delete：删除一个对象的属性或者一个数组中某个键值。

    • 删除成功后，属性或者元素会变成undefined
    • delete删除数组元素，元素就不在数组中了，寻址返回undefined。

  • typeof：返回一个表示 operand 类型的字符串值。

  • void：表明一个运算没有返回值。

关系运算符

对操作数进行比较，根据比较结果真或假，返回相应的布尔值。

  • in：如果指定的属性存在于所指定的对象中，返回true

  • instanceof：如果所判别的对象确实是所指定的类型，则返回true

表达式

表达式是一组代码的集合，它返回一个值。

js的表达式类型：

  • 算数：得出一个数字，例如 3.14159。（通常使用算数运算符）
  • 字符串：得出一个字符串，例如，”Fred” 或 “234”。（通常使用字符串运算符)。）
  • 逻辑值：得出 true 或者 false。（经常涉及到逻辑运算符)。）
  • 基本表达式：javascript 中基本的关键字和一般表达式。
  • 左值表达式：分配给左值。

基本表达式

  • this 关键字：被用于指代当前的对象
  • 分组操作符 () ：控制了表达式中计算的优先级。

左值表达式

  • new 运算符：创建一个自定义类型或者是预置类型的对象实例。

  • super 关键字：调用一个对象父类的函数。

1.6数字和日期

数字

js内的数字均为双精度浮点类型（介于±2^−1023 和 ±2^+1024 之间的数字）。

整数数值仅在 ±(2^53 - 1) 的范围内可以表示准确。

数字类型还能表示三种符号值：

  • + Infinity（正无穷）
  • - Infinity（负无穷）
  • NaN (not-a-number，非数字)

可以使用四种数字进制：二、八、十、十六。

  • 二进制数字：0b（0B）开头。
  • 八进制数字：0（0o、0O）开头。假如0后的数字不在0-7的范围内，转换为十进制。
    • ES5严格模式下进制使用八进制语法（因为八进制不是ES5规范的一部分，但数字前加0就可以被所有浏览器支持：0644 === 420 而且 `”\045” === “%”）。
    • ES6使用八进制，使用前缀”0o“
  • 十六进制数字：0X（0x）开头。

Number对象

内置的Number对象有一些有关数字的常量属性，不可更改，但可以使用。

  • 属性

    • Number.MAX_VALUE —— 最大值

    • Number.MIN_VALUE ——最小值

    • Number.NaN —— 非数字

    • Number.POSITIVE_INFINITY —— 正无穷，溢出时返回

    • Number.NEGATIVE_INFINITY —— 负无穷，溢出时返回

    • Number.MIN_SAFE_INTEGER、Number.MAX_SAFE_INTEGER —— JavaScript 最小/最大安全整数。

    • Number.EPSILON —— 表示 1 与大于 1 的最小浮点数之间的差值。

      • ```js
        const result = Math.abs(0.2 - 0.3 + 0.1);
        console.log(result < Number.EPSILON);
        // Expected output: true

    • toFixed() —— 使用定点表示法来格式化该数值（小数点保留位数）。

    • toPrecision() —— 返回一个指定精度的数字字符串。

Math对象

内置的Math数学常项和函数也有一些属性和方法。

例如PI属性、sin()方法等。

  • abs() —— 绝对值
  • floor(),ceil() —— 返回 小于等于/大于等于 参数的最大整数
  • random() —— 返回0和1之间的随机数
  • sqrt() —— 平方根
  • round() —— 四舍五入后最接近的整数
  • 更多，详见Math。

日期对象

js没有日期数据类型，但可以使用Date对象和其方法来处理日期和时间。

Date对象则基于 Unix Time Stamp，即自 1970 年 1 月 1 日（UTC）起经过的毫秒数。

创建一个日期对象：

  • 参数parameters的格式：
    • 无参数。例如今天today = new Date();.
    • 一个符合以下格式的表示日期的字符串：”月 日，年 时：分：秒”。例如： var Xmas95 = new Date("December 25, 1995 13:30:00")。如果你省略时、分、秒，那么他们的值将被设置为 0。
    • 一个年，月，日的整型值的集合，例如： var Xmas95 = new Date(1995, 11, 25)。
    • 一个年，月，日，时，分，秒的集合，例如： var Xmas95 = new Date(1995, 11, 25, 9, 30, 0);.

Date对象的方法分为以下几类：

  • set：设置
  • get：获取
  • to：返回Date对象字符串格式的值
  • parse和UTC方法：解析Date字符串

示例：返回当前时间的数字时钟

1.7文本格式化

字符串

JS 中的 String 类型用于表示文本型的数据。它是由无符号整数值（16bit）作为元素而组成的集合。

字符串内的每个元素在字符串中占据一个位置。index从0开始。

String字面量

  • 可以使用单引号或双引号创建简单的字符串
  • 转义序列创建更复杂的字符串
    • \x —— 十六进制
    • \u —— unicode

String对象

String对象是对原始string类型的封装。

可以在String字面值上使用String对象的方法，因为JS会自动把String字面值转换为一个临时的String对象，调用其方法后再丢弃。

除非必要，应该尽量使用String字面值。

因为String对象的某些行为可能并不符合预期。

String对象含有许多方法，例如：

  • CharAt()：返回一个由给定索引处的单个 UTF-16 码元构成的新字符串。
  • indexOf(), lastIndexOf() ：返回字符串中指定子串的位置/最后位置。
  • **concat()**：连接字符串
  • toLowerCase(), toUpperCase() —— 分别返回字符串的小写表示和大写表示。
  • match, replace, search —— 通过正则表达式来工作。

多行模板字符串

模板字符串是一种允许内嵌表达式的 String 字面值。可以用它实现多行字符串或者字符串内插等特性。

模板字符串使用反勾号 (``` `) 包裹内容。模板字符串可以包含占位符。占位符用美元符号和花括号标识 (${expression}).

多行

使用模板字符串，可以多行显示且无需插入转义符。

字符串嵌入表达式

模板字符串，可以使用语法糖让嵌入功能的实现代码更具可读性。

Intl国际化

Intl 对象是 ECMAScript 国际化 API 的一个命名空间，它提供了精确的字符串对比、数字格式化，和日期时间格式化。

  • 日期时间格式化：**Intl.DateTimeFormat**对象使日期和时间在特定的环境下格式化。

  • 数字格式化：**Intl.NumberFormat** 对象能使数字在特定的语言环境下格式化。

  • 定序：**Intl.Collator** 对象用于语言敏感的字符串比较。

1.8正则表达式

点此查看有关正则表达式的笔记。

在js中，正则表达式也是对象。

创建正则表达式（两种方法）：

  1. /正则表达式字面量/
  2. RegExp对象的构造函数：new RegExp()

使用正则表达式：

方法 描述
exec 一个在字符串中执行查找匹配的 RegExp 方法，它返回一个数组（未匹配到则返回 null）。
test 一个在字符串中测试是否匹配的 RegExp 方法，它返回 true 或 false。
match 一个在字符串中执行查找匹配的 String 方法，它返回一个数组，在未匹配到时会返回 null。
matchAll 一个在字符串中执行查找所有匹配的 String 方法，它返回一个迭代器（iterator）。
search 一个在字符串中测试匹配的 String 方法，它返回匹配到的位置索引，或者在失败时返回 -1。
replace 一个在字符串中执行查找匹配的 String 方法，并且使用替换字符串替换掉匹配到的子字符串。
split 一个使用正则表达式或者一个固定字符串分隔一个字符串，并将分隔后的子字符串存储到数组中的 String 方法。

使用选择：

  • 只是想知道能否有匹配 —— test、search
  • 想要匹配并获得更多信息 —— exec、match

通过标志进行高级搜索

正则表达式有六个可选参数 (flags) 。这些参数既可以单独使用也能以任意顺序一起使用，并且被包含在正则表达式实例中。

标志 描述
g 全局搜索。
i 不区分大小写搜索。
m 多行搜索。
s 允许 . 匹配换行符。
u 使用 unicode 码的模式进行匹配。
y 执行“粘性 (sticky)”搜索，匹配从目标字符串的当前位置开始。

语法：

  • var re = /pattern/flags;
  • var re = new RegExp("pattern", "flags");

1.9索引集合类

按索引值排序的数据集合。包括数组和类数组结构，如 Array 对象和 TypedArray 对象。

数组

数组是由名称和索引引用的值构成的有序列表。

js中没有明确的数组数据类型。可以使用预定义的Array对象及其方法来处理数组。

创建数组：

  • Array()内只有单个Number时，会被解释为arrayLength。但必须为整数，否则报错。

创建长度不为0，但没有任何元素的数组：

数组也可以被赋值为新对象或现有对象的属性。

引用数组元素：因为元素也是属性，可以使用属性访问器来访问。

  • 属性访问器语法

在实现层面，JavaScript实际上是将元素作为标准的对象属性来存储，把数组索引作为属性名。

遍历数组

  • 使用for循环，length作为判断条件

  • for循环，但以下标查看元素是否存在作为循环条件（更高效）

  • **forEach()**方法。

    • 未赋值的不会再forEach循环迭代。但手动赋值为undefined会被列出。

  • 由于JavaScript被保存为标准对象属性，因此不建议使用for…in循环遍历。因为普通元素和所有可枚举属性都将被列出。

数组方法

  • concat() ：连接两个或多个数组，并返回一个新的数组

  • join() ：将数组中的所有元素连接成一个字符串。参数为分割符。

  • push() ：数组末尾添加元素，返回length。

  • unshift() ： 在数组开头添加元素，返回length。

  • pop() ：从数组中移出最后一个元素，并返回该元素。

  • shift() ：从数组中移出第一个元素，并返回该元素。

  • slice() : 从数组中提取一个片段，并作为新数组返回。参数为开始，结束索引。

  • at() : 返回指定索引出的元素。索引超出范围，返回undefined。

  • splice() ：移出一些元素，[并替换它们]，返回从数组中删除的元素

  • reverse() ：翻转数组。

  • flat() ：返回一个新的数组，并根据指定深度递归地将所有子数组元素拼接到新的数组中。

  • sort() : 数组排序。可以接受回调函数来决定如何比较数组元素

  • indexOf() ：数组中搜索参数元素，并返回第一个匹配的结果。正向搜索。

  • lastIndexOf() ：和indexOf类似，但反向搜索。

  • 以下为迭代方法。

  • forEach() : 对数组中的每个元素执行 callback 并返回 undefined。

  • map() : 返回由每个数组元素上执行 callback 的返回值所组成的新数组。

  • flatMap() : 先执行 map()，再执行深度为 1 的 flat()

  • find() ：方法返回 callback 返回 true 的第一个元素。

  • findLast() ：返回 callback 返回 true 的最后一个元素。

  • findIndex() ：返回 callback 返回 true 的第一个元素的索引。

  • findLastIndex() ：返回 callback 返回 true 的最后一个元素的索引。

  • every() ：如果如果 callback 对数组中的每一个元素都返回 true，则返回true

  • some() ：如果 callback 对数组中至少一个元素返回 true，返回true。

  • reduce() ：对数组中的每个值执行 callback(accumulator, currentValue, currentIndex, Array)。将列表中的元素减少到单个值。返回 callback 函数返回的最终值。

  • reduceRight() ：工作原理类似于 reduce()，但从最后一个元素开始。

稀疏数组

数组可以包含空槽（不是用undefined填充）。

但在某些操作里，空槽就像是被填入了undefined一样。

例如索引访问空槽、for…of、展开运算，都返回undefined。

而且在其他方法，特别是迭代方法，空槽直接被跳过。

多维数组

数组可以嵌套，所以可以创建多维数组。

类数组对象

一些JavaScript对象（如 document.getElementsByTagName() 返回的 NodeList 或 arguments 等 JavaScript 对象），有与数组相似的行为。

但并不共享数组的所有方法，直接调用会报错。

可以通过Function.prototype.call() 间接调用它们。
